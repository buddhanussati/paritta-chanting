## Paritta Chanting

This is a web-based app & tool designed to support practitioners in listening to, learning, and reciting the Paritta Protection Suttas on a daily basis. It features synchronized text & audio chanting for precise follow-along practice, includes memorization drills by hiding random words, and an interactive dashboard to track memorization progress.

### 4. Mastery Dashboard & Progress

Located above the text in Recitation Mode, this dashboard visualizes your memory retention.

* **Heatmap Grid:** Each box represents a line of text.
* **Gray:** Not memorized.
* **Green:** Memorized.


* **Marking Progress:** When a line is active, a **"Memorized"** checkbox appears below the text. Check this to mark the line as complete (100%).
* **Statistics:**
* **Section Progress:** Completion % of the current Sutta.
* **Memorized Total:** Completion % of all the Paritta Suttas.


* **Achievements:** You will receive a "Sutadhara" banner for completing a sutta, and a "Dhammadhara" Medal Modal for completing the entire text.
